1. Salary Predictor
   
A Machine Learning project that predicts employee salary based on years of experience and education level.

2.Technologies Used
- Python
- Pandas
- Scikit-learn (Linear Regression)

3.How It Works
- The model learns patterns from historical salary data
-  The user provides their experience and education level as input
- The model predicts the expected salary based on the learned pattern

4. Features
- Train/Test split for model evaluation
- Real time prediction system
- Simple and interpretable Linear Regression model

5. Future Improvements
- Add more features (location, job role, company size)
- Try different algorithms for comparison
- Expand dataset for better accuracy
